## 👨‍💻 Sobre mim

Sou desenvolvedor Full Stack com foco na construção de aplicações web completas, organizadas e preparadas para evoluir.

No frontend, desenvolvo interfaces responsivas com **React**, **Tailwind CSS**, **HTML** e **CSS**. No backend, construo APIs com **Python** e **FastAPI**, utilizando **PostgreSQL** e **SQLAlchemy** para persistência de dados.

Também trabalho com **Docker**, **Redis**, **Celery** e **workers**, criando fluxos assíncronos, processamento em segundo plano e soluções mais eficientes para tarefas pesadas.

Valorizo código limpo, organização, responsabilidade técnica e decisões que tornem o sistema mais seguro, performático e fácil de manter.

## 🏗️ Como estruturo minhas aplicações

```text
Usuário
   │
   ▼
React + Tailwind CSS
   │
   ▼
FastAPI
   │
   ├──────────────► PostgreSQL
   │                    │
   │                SQLAlchemy
   │
   └──────────────► Redis
                        │
                        ▼
                  Celery + Workers
```

Essa estrutura permite separar responsabilidades, manter o código organizado e executar tarefas pesadas sem bloquear a experiência do usuário.

## ✅ Princípios que aplico

- Componentização e reutilização no frontend
- APIs organizadas e bem definidas
- Separação de responsabilidades

- Uso de cache quando existe ganho real de performance
- Containers para padronizar ambientes
- Código legível, previsível e fácil de manter
