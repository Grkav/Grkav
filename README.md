## 🏗️ Conceitos que estudo e aplico

- Arquitetura monolítica e microsserviços
- Escalabilidade horizontal e vertical
- Balanceamento de carga em camadas L4 e L7
- Cache com Redis
- Filas, workers e processamento assíncrono
- Idempotência em operações críticas
- APIs RESTful e autenticação
- Docker e Docker Compose
- Bancos de dados relacionais
- Deploy em máquinas virtuais e serviços gerenciados
- AWS EC2, ECS, Lambda, RDS e EBS
- Azure Virtual Machines, Container Apps e bancos gerenciados
- CI/CD e automação de deploy
